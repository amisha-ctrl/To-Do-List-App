📝 To-Do List App
A simple and interactive To-Do List web application built with HTML, CSS, and JavaScript. This app allows users to add, complete, and delete tasks easily, helping to manage daily activities efficiently.

🚀 Features
Add new tasks dynamically.
Mark tasks as complete.
Delete tasks from the list.
Simple and clean user interface.
Responsive design for mobile and desktop.

🛠️ Technologies Used
HTML
CSS
JavaScript
FontAwesome
Google Fonts
